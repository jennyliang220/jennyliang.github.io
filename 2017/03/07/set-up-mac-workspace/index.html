MAC前端工具安装和环境搭建-写在某前端工程师格式化电脑后

Posted by JennyL on 2017-03-07
JennyL前端经验集
  • Home

为了使用sketch，升级了mac系统版本os 10.12.1，然后各种应用大崩溃：QQ, safari, 邮件，印象笔记都打不开了。于是重新格式化电脑安装了10.11……一切正常之后，开始配置常用的前端工具。

1. node &npm


node和npm几乎是一切的基础。有了npm包管理工具，就能安装接下来的工具了。
下载安装包：https://nodejs.org/en/

2. sublime


每天写代码的地方。如果不喜欢vim，sublime绝对是零基础入手的神器。
下载安装包：http://www.sublimetext.com/
需要配合package control使用，大大增强功能。

3. iTerm2


好用的控制台, 代替mac自带终端。
下载安装包：http://www.iterm2.com/

4. git & github


文件版本管理，连接github
下载安装包：https://git-scm.com/download/mac
配置ssh key: https://github.com/settings/keys

5. brew & make

brew: 包管理工具，可继续安装nginx，make, wget
复制命令安装：https://brew.sh/
新建Makefile, 使用make快捷执行命令

6. Xcode


安装brew的时候要求安装xcode，那就安吧。4.55GB也太大了吧

到这里，前端工具基本已经安装完毕。另外贴几个设计+交互软件的地址备份

7. Sketch


现在已经不用ps和ai了，sketch实在是好用。
下载安装包：https://www.sketchapp.com/
常用插件：sketch measure; style inventory

8. Tumult Hype

交互设计工具
下载安装包：http://tumult.com/hype/download/ （需要翻墙）

Adobe XD


看在adobe家族如此力挺这个设计+交互工具，还是值得一试的。
网盘下载：http://pan.baidu.com/s/1jHqMWYi
